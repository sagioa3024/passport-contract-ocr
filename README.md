# MVP: договор по данным паспорта

Проект рассчитан на обычный облачный хостинг с PHP.

## Что уже есть

- загрузка фото/скана паспорта;
- поддержка нескольких изображений и ксерокопий, где паспортные страницы расположены на одном листе;
- ручная форма проверки данных;
- распознавание через OpenAI API, если задан ключ;
- создание договора в формате DOCX на основе шаблона `contract_template.docx`;
- базовая защита папок с конфигом, загрузками и готовыми файлами.

## Установка на Beget

1. Загрузите содержимое папки `public_html` в `public_html` домена на Beget.
2. Скопируйте `config.example.php` в `config.php`.
3. В `config.php` укажите OpenAI API-ключ:

```php
'openai_api_key' => 'sk-...',
```

4. Откройте сайт в браузере.

Если ключ не указан, приложение все равно работает: данные можно ввести вручную и скачать договор.

## Файлы, которые нельзя коммитить

- `public_html/config.php` — содержит OpenAI API ключ.
- `public_html/uploads/*` — временные загруженные документы.
- `public_html/generated/*` — готовые договоры.

## Важно

Паспортные данные относятся к персональным данным. Перед реальным запуском нужно добавить политику обработки персональных данных, согласие пользователя и понятный срок удаления файлов.
